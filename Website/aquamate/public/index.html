How to use

  1. Power up the Aquamate device.
  2. Connect to the desired local network.
    1. Use any Wi-Fi enabled device(Ex: Smart phone, laptop) and turn on Wi-Fi.
    2. Search available Wi-Fi networks.
    3. Connect to the "Aquamate tank {id}" with the password "11111111".
    4. Go the IP address "10.0.1.1" or use the URL http://10.0.1.1
    5. Click on "Configure WiFi"
    6. Select desired WiFi network & enter password.
    7. Click on "Save"
  3. Now you click on above relevent tank to view it's details.

Note: You have to do this only once, unless there is a password change in desired network.

Project Details

ESP32 Web Server

After connect to the local network, a web server will run in ESP32 chip & the data will be updated realtime.

There are 2 PCBs inside the Main enclosure.
